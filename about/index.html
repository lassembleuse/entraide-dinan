A propos

Ce site a pour ambition de centraliser et diffuser les actions d'entraide sur le territoire de Dinan (Dinan agglomération) durant l'épidémie de coronavirus (covid-19).

C'est, pour le moment, une initiative individuelle d'une dinanaise développeuse web. Vous êtes les bienvenus pour prêter main forte !

Pour proposer des données : questionnaire en ligne

Pour toute autre contribution : bonjour@lassembleuse.fr
